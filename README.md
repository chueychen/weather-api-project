# AI-Powered Weather Outfit Recommendation System

This project retrieves weather data using the OpenWeather API and generates outfit recommendations using the Gemini API.

## Features
- Convert city names into geographic coordinates (Geocoding API)
- Retrieve real-time weather forecast data
- Extract key metrics (temperature, humidity, wind speed)
- Generate AI-based outfit suggestions

## Technologies
- Python
- OpenWeather API
- Gemini API
- Requests
- Environment variables (.env)
